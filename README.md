# 가계부 앱 (Budget App)
> Flutter + FastAPI를 이용한 개인 가계부 관리 앱

## 📱 주요 기능
- 지출 내역 관리
- 카테고리별 분류
- 실시간 데이터 동기화

## 🛠 기술 스택
- Frontend: Flutter (Dart)
- Backend: FastAPI (Python)
- Database: SQLite
